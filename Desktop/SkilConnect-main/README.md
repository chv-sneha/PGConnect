# SkillConnect

## About

SkillConnect is a comprehensive platform designed to bridge the gap between learning and earning. It empowers individuals to discover their strengths, exchange skills with peers, and unlock local opportunities.

## Features

- **CareerLens**: AI-powered career guidance to discover your strengths and map your perfect career path
- **Skill Exchange**: Learn from peers and teach what you know in a community of mutual growth
- **SkillLink**: Connect with local opportunities, offer your skills, find gigs, and earn in your community
- **MentorConnect**: Get guidance from experienced professionals and validate your skills
- **Verified Portfolio**: Earn digital badges and build a portfolio showcasing your proven abilities
- **Enterprise Bridge**: Connect with businesses and MSMEs looking for skilled talent

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn-ui
- **Routing**: React Router
- **State Management**: React Query
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd skill-link-grow
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
